# Installation
[Node.js](https://nodejs.org) Make sure you have version 14+ installed. Then run the following:

```bash
npm install -g genlic@latest
```
or
```bash
yarn global add genlic@latest
```

# Usage
```bash
genlic
```
or
```bash
npx genlic
```
